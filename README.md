Welcome to TranscriptStudio
===========================

This Windows desktop application (requires .NET framework to run) provides handy features for people that write transcripts of media file.
It supports keyboard navigation in media files for:

- start/pause/resume 
- fast forward / slow forward
- skip 3 sec, skip 10 sec
- go back 3 sec, go back 10 sec

Transcripts are saved to XML file, which can be also opened with text editor. Application also supports export SRT (subtitle) and XLS (MS Excel) file formats.
